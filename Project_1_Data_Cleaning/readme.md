# Week 1: Data Cleaning and Preparation

## Project Overview
This project focuses on auditing and cleaning the raw E-Commerce Orders Dataset to ensure accuracy and structural integrity before performing any visual analysis.

## Key Actions Taken:
- Checked for duplicates using the Excel 'Remove Duplicates' tool (0 duplicates found).
- Handled missing values by replacing 309 blank cells in the `Coupon Code` column with 'No Coupon'.
- Standardized data formatting across dates and verified pricing mathematics (`Quantity` × `Unit Price` = `Total Price`).

1. `Cleaned_Orders_Dataset_AapkaNaam.xlsx` - Cleaned master workbook with a detailed audit log on Sheet 2.
2. `Data_Cleaning_Report_AapkaNaam.pdf` - Complete documentation with step-by-step screenshots.
